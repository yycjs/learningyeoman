About Generators...

Generators are yeoman (nodejs) apps that scaffold out (generate) new apps

In our case, that's an HTMl5 app

When you make a new generator, you are using a generator

Ok... Now for the fun

Our Goal (HTML5 App)

We will create a new yeoman app to automatically generate the following for us, saving us time in the future:

  • script tags, so we can automatically specify which JS files to use in a given project
  • style tags, so we can import whatever front-end framework we want
  • Author and project information, including licensing details, which will be appended to the beginning of each file, in a block-style comment
